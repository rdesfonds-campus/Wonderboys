# Wonderboys 🛡️⚔️

Jeu de plateau inspiré de l’univers **Donjons & Dragons**, développé en **Java**.  
Un personnage avance sur un plateau de **64 cases** en lançant un dé virtuel. 

(Version 2.0 sans God Classe)

## Fonctionnalités
- Création d’un personnage (**Warrior** ou **Wizard**) avec nom, points de vie et force d’attaque. 
- Gestion d’un plateau de 64 cases avec déplacement via un dé virtuel. 
- Menu texte pour créer, afficher, modifier le personnage et lancer la partie. 
- Affichage de la progression du joueur case par case jusqu’à la fin du plateau. 

## Technologies utilisées
- **Langage** : Java
- **Paradigme** : Programmation orientée objet (classes, héritage, classes abstraites). 
- **Outils** : IntelliJ IDEA, UML (diagramme de classes) via draw.io ou Umletino. 

## Prérequis
- Java installé (JDK 17 ou version compatible).
- Un IDE Java (IntelliJ, Eclipse, VS Code avec extension Java).
- Git installé si tu utilises GitHub.
